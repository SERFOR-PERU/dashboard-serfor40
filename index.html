🌐 Publicar y Compartir Dashboard

1 Descarga el dashboard con todos los datos actuales

Genera un archivo .html autocontenido con el estado actual. Se abre directamente en cualquier navegador.

Incluye todos los proyectos, costos, avances, recursos y selecciones actualizadas.

2 Publica en la web para obtener un enlace compartible

🟢 Netlify Drop Gratis · Sin registro · Recomendado
  1. Abre app.netlify.com/drop
  2. Arrastra el archivo al área de carga
  3. Obtén enlace: https://nombre-aleatorio.netlify.app
  4. Comparte — se abre en Chrome o Edge
🔵 SharePoint / OneDrive SERFOR Red institucional
  1. Sube el archivo a SharePoint o OneDrive
  2. Haz clic derecho → Compartir → Cualquier persona con el enlace
  3. Copia y comparte el enlace

3 Sincronizar actualizaciones

⬆ Importar estado (.json)

El .json contiene solo modificaciones y pesa muy poco. Envíalo por correo institucional.

⬆ Actualizar Estado de Proyectos

Sube un CSV para actualizar múltiples proyectos. Columnas aceptadas:

N°, Avance, Estado, Programadores, Analistas, Costo, Responsable, Acciones, Observaciones, Limitaciones
📂 Arrastra tu CSV aquí o haz clic para seleccionar

El campo N° es obligatorio como identificador.
